# homebrew-lab

Personal Homebrew tap for [@ibrarwtf](https://github.com/ibrarwtf)'s Mac apps.

## Install

```bash
brew tap ibrarwtf/lab
brew install --cask claude-limits
```

## Casks

| Name | What | Source |
|---|---|---|
| `claude-limits` | Native macOS menu-bar app showing your Anthropic usage at a glance | [ibrarwtf/claude-limits](https://github.com/ibrarwtf/claude-limits) |
